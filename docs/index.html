Data Aggregator

A FusionCharts Time Series Extension

This extension will let the user to aggregate his raw data based on various standard aggregation methods and get the visualization of the aggregated data in FusionCharts time-series chart.

Browser Inclusion

<script type="text/javascript" src="js/fusioncharts.min.js"></script> <script type="text/javascript" src="js/fcts-ext-dataaggregator-es6.js"></script>

Node Inclusion

var FusionCharts = require('fusioncharts'); var DataAggregator = require('data-aggregator')(FusionCharts);
